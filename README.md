# DigiTools

A modern platform to browse and purchase premium digital tools.

## About

DigiTools lets users explore premium digital products, add them to cart, and manage purchases with real-time updates.

## Technology Used

- React 19
- Tailwind CSS
- DaisyUI
- React Toastify
- Lucide Icons
- Vite

## Features

1. Shopping Cart - Add or remove items, cart count updates in real time, cart saves after page refresh

2. Responsive Design - Works on mobile, tablet, and desktop screens using Tailwind and DaisyUI components

3. Toast Notifications - Get alerts when you add, remove, or clear cart items
